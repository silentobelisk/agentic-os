What's inside

Plan Usage

Auto-detects your Claude plan and shows live session + weekly windows with reset countdowns. No API cost — it reads what you already pay for.

Knowledge Graph

Point it at a folder of notes and it maps your second brain into an interactive force-directed graph, built locally with /graphify.

Live Terminal

Talk to Claude Code inside the console — a streamed session on your subscription, no API key required.

Most-used Skills

Your skills ranked by how often you invoke them. Click to run inline or launch a real terminal session.

Audience Signals

Track YouTube, Instagram & TikTok follower counts — free, no API keys, fetched from your machine.

Industry Signal

A live board of AI/dev numbers from Hacker News & GitHub, refreshed on demand.

Install

01

Download

Grab the .dmg with the button above (Apple Silicon).

02

Drag to Applications

Open the disk image and drag Agentic OS into your Applications folder.

03

First open

Not notarized yet, so macOS guards it. Open it once, then in System Settings → Privacy & Security click Open Anyway.

Still blocked? Open Terminal and run xattr -dr com.apple.quarantine "/Applications/Agentic OS.app", then open the app normally — this clears the download quarantine in one step.

On first run, a one-time setup walks you through naming your operator profile, confirming your Claude plan, and (optionally) connecting your second brain. Nothing leaves your device.
